# ADMIN DASHBOARD

## Purpose
Central control panel for managing the entire school system.

---

## Header
- Title: Dynamic (Dashboard / Students / Teachers / etc.)
- Search: Students, Teachers, Classes
- Primary Action: Dynamic (Add Student / Add Teacher / Create Class)
- Notifications:
  - New student registered
  - Grades submitted
  - Announcements created
- Profile: Avatar only
