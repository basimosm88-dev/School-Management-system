# 🧠 TYPOGRAPHY SYSTEM — SCHOOL MANAGEMENT SYSTEM

## 🎯 PURPOSE

This file defines ALL typography rules for the system.

It ensures:

* consistency
* readability
* professional UI
* no random font behavior

---

# 🔤 FONT FAMILY

Primary Font:

"Mona Sans", sans-serif

Fallback:

system-ui, sans-serif

---

# 🔠 FONT WEIGHTS

| Usage            | Weight  |
| ---------------- | ------- |
| Body text        | 400     |
| Labels           | 500     |
| Buttons          | 500–600 |
| Section titles   | 600     |
| Important values | 700     |

---

# 🔡 FONT SIZE SCALE

## Base System

| Type           | Size | Line Height |
| -------------- | ---- | ----------- |
| Small text     | 12px | 1.4         |
| Labels         | 13px | 1.4         |
| Body text      | 14px | 1.5         |
| Secondary text | 13px | 1.5         |

| ------- | ---- | ----------- |

## KPI / Numbers

| Type      | Size    | Weight |
| --------- | ------- | ------ |
| KPI Value | 22–26px | 700    |

# ✨ LETTER SPACING

| Element   | Letter Spacing |
| --------- | -------------- |
| Body text | 0.01em         |
| Labels    | 0.02em         |
| Headings  | -0.01em        |
| Buttons   | 0.02em         |

---

# 📏 LINE HEIGHT RULES

* Body text → 1.5 – 1.6
* Headings → 1.2 – 1.3
* Tables → 1.4
* Buttons → 1.3

---

# 🔤 TEXT CASING RULES

## Titles

Use: Capitalized Words
Example:

* "Student Profile"
* "Attendance Report"

---

## Labels

Use: Sentence case
Example:

* "Phone number"
* "Registration date"

---

## Buttons

Use: Capitalized Words
Example:

* "Add Student"
* "Save Changes"

---

## Table Headers

Use: UPPERCASE or Title Case (choose ONE and apply globally)

Recommended:

* Title Case

Example:

* "Full Name"
* "Student ID"

---

# 📊 TABLE TYPOGRAPHY

* Font size: 13–14px

* Weight:

  * Header → 600
  * Body → 400

* Row spacing:

  * comfortable (not tight)

---

# 🧾 FORM TYPOGRAPHY

## Labels

* 13–14px
* Weight: 500

## Inputs

* 14px
* Padding must allow breathing room

## Placeholder

* lighter color
* not bold

---

# 🔘 BUTTON TYPOGRAPHY

* Font size: 14px
* Weight: 500–600
* Letter spacing: 0.02em

---

# 📌 SIDEBAR TYPOGRAPHY

* Menu items:

  * 14px
  * Weight: 500

* Section labels:

  * 12px
  * Uppercase
  * Slight letter spacing

---

# 📊 DASHBOARD TYPOGRAPHY

## KPI Cards

* Label:

  * 12–13px
  * Weight: 500

* Value:

  * 22–26px
  * Weight: 700
